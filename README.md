📊 Data Leverager – Power Query Transformation Project

📌 Project Overview

This project demonstrates the ETL (Extract, Transform, Load) process using Power BI – Power Query Editor.

❌ No Data Visualization

✅ Focus on Data Cleaning & Transformation

🖼️ Step-by-Step Workflow

## 🔹 Step 1: Project Introduction

Project Name: Data Leverager

Tool Used: Power BI (Power Query)

Objective: To extract, clean, transform, and consolidate data from multiple sources into a structured dataset

## 🔹 Step 2: Data Extraction

Data Sources Loaded:

Web (HTML Table)

Excel Files (January, February, March)

Employee Dataset

## 🔹 Step 3: Data Cleaning (Basic Transformations)

Operations Performed:

Removed null and blank values

Removed duplicate records

Promoted first row as headers

Renamed columns for clarity

Corrected data types (Text, Number, Date)

## 🔹 Step 4: Text Transformation

Functions Applied:

Text formatting: UPPER(), LOWER()

Cleaning: TRIM(), CLEAN()

Value replacement using REPLACE()

Splitting columns based on delimiters

##🔹 Step 5: Numeric Transformations

Tasks Completed:

Rounded numeric values to 2 decimal places

Created calculated column:

👉 Profit = Revenue – Cost

## 🔹 Step 6: Date & Time Transformation

Derived Columns:

Day

Month

Year

Quarter


Advanced Calculations:

Fiscal Month

Age calculation from Birthdate

## 🔹 Step 7: Conditional Columns

Sales Categorization:

High → Sales > 10,000

Medium → 5,000 to 9,999

Low → Sales < 5,000

## 🔹 Step 8: Pivot & Unpivot Operations

Pivot: Convert rows into columns

Unpivot: Convert columns into rows

## 🔹 Step 9: Data Integration

Operations Performed:

Merged dataset with Employee data

Appended monthly files (Jan, Feb, Mar) into a single dataset

## 🔹 Step 10: Grouping & Aggregation

Grouped by Region:

Total Sales

Average Order Value

Transaction Count

## 🔹 Step 11: Data Profiling & Quality Check

Tools Used:

Column Profile

Column Distribution

Column Quality

## ✅ Final Output

Cleaned & structured dataset

Ready for reporting and analysis

Improved data quality and consistency
